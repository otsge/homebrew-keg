# OTSGE Keg

## How do I install these casks?

`brew install --cask otsge/keg/<cask>`

Or `brew tap otsge/keg` and then `brew install --cask <cask>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/keg"
cask "<cask>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
